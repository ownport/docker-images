# Builder

## Docker Tools
```
 % ./builder docker-tools --help       
usage: buildtools docker-tools [-h] [--list-containers] [--remove-exited-containers] [--remove-all-containers] [--list-images] [--remove-none-images] [--remove-all-images]

optional arguments:
  -h, --help            show this help message and exit
  --list-containers     list containers
  --remove-exited-containers
                        remove exited containers
  --remove-all-containers
                        remove all containers
  --list-images         list images
  --remove-none-images  remove none images
  --remove-all-images   remove all images
```
